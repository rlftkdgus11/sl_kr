# sl_kr

FNAF Sister Location Korean patch project snapshot.

## Included
- patch_src/ core dsound proxy source and build/scripts
- runtime mapping files (sound_hash_map.tsv, subtitles_by_*.txt, sound_names.csv)
- Korean subtitle JSON files
- mobile_subs/ timeline fragments
- baseline runtime config (hash_only.txt)

## Notes
- Large game binaries/assets are intentionally excluded.
- This repository focuses on patch logic/data.
